Waste Sorting Game

Play the EcoSort game and learn about the types of waste and which bin it belongs in.

Play EcoSort Game

Instructions

1. Click the button to open the game.
2. Decide which bin the item should go in.
3. Earn points for every correct answer!

Waste Classifier

Use the Teachable Machine model to classify and understand different waste items.

Try Waste Classifier

Instructions

1. Click the button to open the classifier.
2. Allow camera access when prompted.
3. Hold up an item to see how it is classified.
